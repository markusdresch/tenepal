# Annotator Screenshots

For the public GitHub draft, include only **youth-safe** annotator screenshots.

## What to Show

- the segment table
- language correction controls
- comparison view between configurations
- harmless dialogue or neutral ambient scenes
- cropped UI regions when possible

## What Not to Show

- visible violence
- visible sexual content or nudity
- torture, execution, blood, or assault scenes
- copyrighted full-frame film stills when a cropped interface screenshot would do

## Recommended Screenshot Sources

Prefer scenes from **La Otra Conquista** that are:

- dialogue-heavy
- visually calm
- non-graphic
- useful for showing Nahuatl vs. Spanish annotation decisions

Use **Hernán** only if the selected frame is clearly non-graphic and legally defensible for a documentation screenshot.

## Suggested Screenshot Set

1. Main annotator view with subtitle list and current segment
2. Correction workflow: predicted language -> corrected language
3. Comparison matrix view across multiple EQ configs
4. Export/result confirmation screen if useful

## Practical Capture Advice

- pause on a neutral frame before taking the screenshot
- crop to emphasize the interface, not the film image
- blur or crop any unnecessary identifying media details if needed
- store screenshots under a dedicated public path such as `docs/images/annotator/`

## Alt Text / Captions

Use factual captions, for example:

- "Annotator interface for reviewing segment-level Nahuatl/Spanish labels"
- "Comparison view showing multiple pipeline outputs for the same subtitle segment"
